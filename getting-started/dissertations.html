# Dissertations

Dissertations are a way showcasing research and innovations from the IHPME community at large. With the dissertations manager you can create new entries attached to students. programs, degree types, and graduating years all of which is displayed on the student research page open in new window.

# Example of research page

# Accessing Dissertations

To access the Dissertations click or hover over the Dissertations link on the left-hand side of the WordPress editor. Here you will be presented with a several options.

# Dissertations editor options

There are many options available within the Dissertations menu, most of which are relegated to administrator access only. Staff and General Faculty members will only have "Add New" and "My Dissertations" available to them.

# Example of WordPress sidebar:

# My Dissertations

Provides a listing of all dissertations currently saved within the WordPress database.

# Add New

Adds a new blank dissertations to the database for editing.

# Programs

Configures the listing of available programs within the WordPress database.

# Degrees

Configures the listing of available degrees available to be associated with the dissertations.

# Program Order

Configurable order of all programs currently available within IHPME.

# Degree Order

Configurable order of all programs currently available within IHPME.

# Import Dissertations

A tool for bulk uploading new dissertations into the WordPress database. The template document for importing students can be found in this section.

# Editing & Creating Dissertations

# Adding a new dissertations

To add a new dissertation, select or hover over the Dissertations menu item on the left-hand sidebar and click "Add New".

# Finding a Dissertations

To find a Dissertation within the WordPress editor, select "My Dissertations" from the Dissertations sidebar menu. This will display a listing of all available profiles in the WordPress editor.

List view of all dissertations:

From the list view of all profiles you can filter the available results by several methods:

  • Search Field: Search for a specific dissertation by name (upper right-hand corner).
  • Dropdown filters: Provides several options to filter by date, program, and degree type.
  • By page: Allows jumping to a specific page or advancing forward/backwards within the listing of profiles.

# Viewing a Dissertation

Below is an example of an existing dissertation within the WordPress editor. Dissertations unlike other profiles are very limited in the amount of editable features. Dissertations have limited fields which can be filled in, this enables a consistent format in the directory listing format. There are several requireds and a few optional entries. All fields entered will be displayed in predefined formatting.

# Editing a Dissertation

# Dissertation Title

The current name of the dissertation can be edited by inputting a name into the first field available below the heading "Edit Student Student Dissertation".

# Issue Date

Input the year date when the dissertation is published/issued.

# Student name

Input the name of the student attached to the dissertation. Note this is freeform text, please be mindful of spelling.

# Program

Select from the dropdown menu the appropriate program the student and dissertation is associated with.

Alternatively you can select from the right hand side panel the program associations

# Dissertation Details

Here you can configure two options, a link to the specific published paper (usually an external source) and the appropriate supervisor(s) associated with the student dissertation.

# Revision history

This display a history of published changes to the attached dissertation with the associate account name and timestamp of when the revision was made.

# Excerpt

The excerpt area allows for a short summary of the dissertation when applicable. This is not a required field to fill in but is encouraged.
