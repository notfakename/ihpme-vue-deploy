# Job Postings

Available job postings relevant to IHPME are all managed within the WordPress database. They can be detailed postings formatted like any other page with the editor or simple titles with descriptions and external links. Job postings can also be grouped into categories based on audience type. When a job posting is created it is available for viewing on the job postings page open in new window.

# Example of job posting page

# Accessing Job Postings

To access the Job Postings click or hover over the Job Postings link on the left-hand side of the WordPress editor. Here you will be presented with a several options.

# Job Postings editor options

There are a few options available within the Job Postings menu, most of which are relegated to administrator access only.

# Example of WordPress sidebar:

# All Job Postings

Provides a listing of all job postings currently saved within the WordPress database.

# Add New

Adds a new blank job posting to the database for editing.

# Job Types

Configures the available categories of job types available within the editor. This is used for categorizing job posts based on audience.

# Editing & Creating Jobs

# Adding a new job

To add a new job, select or hover over the Job Postings menu item on the left-hand sidebar and click "Add New".

# Finding a Job Posting

To find a Job Posting within the WordPress editor, select "All Job Postings" from the Job Postings sidebar menu. This will display a listing of all available profiles in the WordPress editor.

List view of all dissertations:

From the list view of all profiles you can filter the available results by several methods:

  • Search Field: Search for a specific dissertation by name (upper right-hand corner).
  • Dropdown filters: Provides several options to filter by date, job type, SEO score, Readability Score
  • By page: Allows jumping to a specific page or advancing forward/backwards within the listing of profiles.

# Editing a Job Posting

# Job Posting Title

The current name of the job posting can be edited by inputting a title into the "Title your new job posting" text area. This title is what will be displayed on the published job posting.

# Posting details or External Link

Bellow the title you can either utilize the area for a link to an external resource by copy and pasting directly into the "Type / to choose a block" area or create a new block for a more detailed description. When pasting a link into the content area the "embed URL" box will automatically appear as an option for inserting into the page, no more actions are required.

# Configuration Panel

With the "job posting title" of the page selected, the configuration panel will present the following controls:

# Available controls

# Summary

The summary area contains basic information about the current page:

  • Visibility: Refers to who can view the page. Default is set to public.
  • Publish: The date on which the job posting was published.
  • URL: Is the address that the job posting is published to the IHPME website.
  • Template: Default template.
  • Post Format: Selector assigning the post type format either "link" or "standard"
  • Author: Assignable author of the job post.

# Opportunity Details

  • Organization: Freeform text for assigning an organization name (often external) to the job posting
  • Closing date: Allows assignment of a closing date for the current job posting
  • Hide closing date: Allows hiding of the closing date of the published job posting

# Yoast SEO

This is a third-party tool that provides helpful guidance in SEO best practices. However, in the context of the job postings this plugin does not provide accurate feedback due to the structure of the page and largely can be ignored.

# Job Types

Allows placement of the job post into one of the predefined categories within the job posting page.

# Excerpt

Optional input area for descriptive text about the job posting.
