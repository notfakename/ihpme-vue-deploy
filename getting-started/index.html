                                      # Getting Started

                                      These documents are written under the assumption that you as the user have an understanding of some WordPress's basic functionalities such as adding blocks to a page, publishing and unpublishing as well as general navigation throughout a WordPress backend.

                                      If additional support is required separate training sessions can be arranged with by contact the appropriate support contacts

                                      # Content Areas

                                      Within the WordPress editor sidebar you will find a listing of various content editors each with a specific purposes for drawing content into other areas of the site. This helps create single entry points of content management and mitigates duplication of content and efforts throughout. The follow content managers area available via the sidebar of the WordPress editor.

                                      Content EditorFunction/Purpose
                                      Alumni StoriesEntries for alumni/graduate stories to be featured throughout the website
                                      AwardsManager for the various awards available at IHPME
                                      CoursesDatabase directory where external and internal courses can be managed
                                      DissertationsInput area for student dissertation information
                                      EventsEvents database manager
                                      Important Dates Date manager for important dates relevant to current students
                                      InitiativesManager for initiatives within IHPME
                                      Job PostingsContent manager for posting of job posting information
                                      Faculty ProfilesContent manager for Faculty profiles
                                      Post-Doc ProfilesContent manager for Post-Doc profiles
                                      Senior Fellow ProfilesContent manager for Senior Fellow profiles
                                      Student ProfilesBasic content manager for student profiles
                                      Staff RolesContent manager for staff roles/profiles
                                      TimetableContent manger for the creation of timetables based on current courses avaiable
                                      PagesDetails on the function of pages within the content of the IHPME website.
                                      Reusable BlocksArea to create reusable elements across multiple pages in the website
