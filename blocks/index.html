                                                    # Getting Started with Blocks

                                                    Blocks are the basic building block of all pages within the WordPress editor. Blocks can generally be placed anywhere on a page. Most blocks also have set of unique options to control their appearance on the page.

                                                    # Adding blocks to a page

                                                    Blocks can be added to the page in the following ways.

                                                    # Block inserter

                                                    In the top left hand corner of the current page you are editing there is a blue plus sign. Clicking this will activate the block gallery where you can navigate all available blocks.

                                                    # Copy & Paste

                                                    Sometimes its easiest just to copy and pate an element from one page to the next. To do so simple highlight an element from another page in the WordPress editor and copy the selected block and simply paste the element into the editor window.

                                                    On Mac:
Copy: ⌘ + C
Paste: ⌘ + V

On Windows:
Copy: Ctrl + C
Paste: Ctrl + V

                                                    # List view

                                                    Sometimes its a little difficult to understand where the current block lives within the overall hierarchy of information. An alternative view to this can be enable by hitting the list view toggle in the upper tool bar of the WordPress editor. Here you can see the list tree of all elements on the current page. Items within in here can be dragged up and down the page, as well removed.

                                                    # Block options

                                                    Universally across the WordPress editor when a block is selected you will find the various additional features and configuration options on the right hand pane. These are unique to each element and are configurable individually.

                                                    # Removing blocks from a page

                                                    # Highlight and delete

                                                    Quick and easy to remove a block is to simple highlight select the element and delete it as you would any text element in a word processor

                                                    # List view

                                                    From the list view selecting the three dot menu on block element will allow you to remove the current selection by selecting "remove.." at the bottom of the menu.

                                                    # Tooltip highlighter

                                                    With the block element selected click the three dot menu in the tool tip highlighter then select "Remove" at the bottom of the list. The full of the "remove" tool will contextually changed based on the selected element.
