# Block Quote

# Block Quote Block Types

The quote block formats a quote into a larger pull quote for greater emphasis. There are two types of block quotes to use within the IHPME WordPress editor. Block quotes can be an effective tool for emphasizing testimonials on a page.

# Adding a block to the page

To add the quote block, toggle the block inserter and search for the block named quote. For more information on adding blocks see Getting Started.

# Editing

To edit the block quote, simply add content to the added block element as you would any text on a page. There are two lines available: the first for the quote, the second for the citation.

Empty block quote example

# Configuration options

When the quote block is selected in the WordPress editor, the right-hand sidebar will display the following configuration options:

  • Style
    • Default: Default style with light grey background colour
    • Bordered: No fill colour. Only 1px outline border surrounding the quote
  • Accent Colour
    • Predefined accent colours from the IHPME colour palette. Applies only the bordered version
