# Points Grid

# Points Grid Block

The points grid is an elegant way of display sequentially ordered information on page. Useful for stepped processes or instructions within a given block content. It can be configured to various amounts of columns to accomodate the content you wish to contain.

# Default

TIP

The point grid will automatically adding sequential numbers when adding/removing more points to the grid

# Using the block in WordPress

# Adding block to the page

To add the points grid block, toggle the block inserter and search for the block named points grid. For more information on adding blocks see Getting Started

# Editing

By default when adding a points grid to a page it contains three columns and three empty blocks for content insertion.

Text can be added as normal to both the title and main content area

More points can be added to the grid by hovering your mouse over the main block element and select the plus sign in the bottom right corner

# Configuration options

With the points grid block selected in the WordPress editior the right-hand sidebar will display the follow configuration options:

  • Columns: Adds or subtracts the amount of columns to the points grid
  • Flexible width toggle: Will reduce number of columns on smaller screens
  • Expand to fit: Stretchitems to fit the first row if not enough items
