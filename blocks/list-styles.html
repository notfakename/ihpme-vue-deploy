# Using the block in WordPress

# Adding block to the page

To add the list block, toggle the block inserter and search for the block named list. For more information on adding blocks see Getting Started

# Editing

Editing lists can be edited like regular content on the page. Each line break is treated as a new list item.

# Configuration options

With the lists block selected in the WordPress editior the right-hand sidebar will display the follow configuration options:

  • Style
    • Default: Standard button style with full colour
    • Outline: No fill colour only 1px outline border surrounding the button
    • Plain: No surrounding fill or border
    • Quick links small: for use when creating quick links to other areas
    • Quick links large: for creating large quick links often in root areas within the website
  • Width Settings
    • 25%: Spans 25% with width of the current container
    • 50%: Spans 25% with width of the current container
    • 75%: Spans 25% with width of the current container
    • 100%: Spans 25% with width of the current container
  • Icon Options
    • Select an icon to accompany the block from the predefined set available.
  • Accent Colour
    • Predefined accent colours from the IHPME colour palette
