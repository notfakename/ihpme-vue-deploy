# Using the block in WordPress

# Adding block to the page

To add the icon block, toggle the block inserter and search for the block named icon. For more information on adding blocks see Getting Started

# Editing

If insert a icon block directly onto the page there are several configuration options available in the tool tip edior. In the tool tip editor you can select the specific icon to use. There are also six sizes available: default, tiny, small, medium, large, and huge. More ofteh than not icons will be used as confirguration options related to a specific block than inserted as a standalone object.

Example of butotn with icon used. [../blocks/buttons.html]

# Configuration options

With the icon block selected in the WordPress editior the right-hand sidebar will display the follow configuration options:

  • Styles
    • Default: Default icon usage with no background or accent colour
    • Buble: Icon will be surrounded by a block of colour
  • Acent Colour: Defines the background colour of the bubble style when selected
